## Getting Started with Plex

Plex lets you stream movies, TV shows, and music from the home server directly in your browser or via the Plex app on any device.

### Creating Your Account

1. Go to [Plex](https://www.plex.tv) and create a free account (or sign in if you already have one).
2. Let me know your Plex account email so I can invite you to the server.
3. Once invited, open Plex and you should see the server appear under **Home**.

### Accessing Content

- **Browser**: Go to [app.plex.tv](https://app.plex.tv) and sign in.
- **Mobile/TV**: Download the Plex app from your app store. Sign in with the same account.
- **Direct**: Use the server link above to open the web player directly.

### Tips

- You can download content for offline viewing in the Plex app (mobile).
- Use the **Watchlist** feature to keep track of things you want to watch.
- Subtitles and audio tracks can be changed mid-stream from the player settings.

## Requesting Content with Seerr

Seerr is a media request tool connected to Plex. If something isn't on the server yet, you can request it and it will be added automatically.

### How to Request

1. Open Seerr using the link above.
2. Sign in with your **Plex account** (click "Sign in with Plex").
3. Search for a movie or TV show.
4. Click **Request** on the result page.

Requests are usually fulfilled within a few hours, depending on availability.

### Request Status

After requesting, you can track the status on the **Requests** page in Seerr:
- **Pending** — Request received, waiting to be processed.
- **Processing** — Being downloaded.
- **Available** — Ready to watch in Plex.

### Notes

- TV show requests: you can request a specific season or the entire show.
- If a request stays pending for more than 24 hours, feel free to reach out.
